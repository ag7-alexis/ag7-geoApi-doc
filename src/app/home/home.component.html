Le but de cette API été de travailler avec une base de données assez importante (+ 5000 enregistrements) afin d'optimiser les requêtes pour garder un temps de réponse le faible possible.

Les données ont été récupérées à partir de l'API découpage administratif du gouvernement ainsi que de l'API de Wikipedia.

Front-End

Le Front-End est fait avec Angular 10 avec la précieuse aide de Bootstrap et de AngularMaterial.

Back-End

La partie Back-End est une API réalisée en Java 11 avec le Framework Spring. J'utilise principalement SpringBoot et SpringSecurity pour sécuriser les requêtes vers l'API avec des JWT.

Base de données

J'utilise une base de données PostgreSQL, la persistance des données est assuré à l'aide de JPA et Hibernate.
