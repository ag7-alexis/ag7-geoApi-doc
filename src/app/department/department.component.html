GET
/department/{param_department} Récupérer des informations sur des départements

L'API retourne la liste des départements, correspondants au paramètre de la recherche (par nom ou numéro de département), de France.

Attribut Description Type
name Nom du département string
number Numéro du département string
info Information sur le département string
region Détails sur la région (détails) correspondate. object
GET
/department/{nom_region}/region} Récupérer les informations des départements d'un region

L'API retourne la liste des départements se situant dans la région correspondante au paramètre de la recherche (nom de la région).

Attribut Description Type
name Nom du département string
number Numéro du département string
info Information sur le département string
region Détails sur la région (détails) correspondate. object
Paramètres

Potentiel paramètres à ajouter à l'url de votre requête envoyé à l'API.

Paramètre Effet
nom_region Le nom complet de la région recherchée.
param_department Le numéro ou le nom (complet ou non) du département recherchée.
noInfo L'API ne retournera pas l'attribut "info".
noRegion L'API ne retournera pas l'attribut "region".
